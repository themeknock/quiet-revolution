Setup, 3 Steps

1

Wispr Flow Download Karo

Download for Mac → Download for Windows →
2

Install + Permissions (Mac)

  • Downloaded .dmg file open karo
  • Wispr icon ko Applications mein drag karo
  • Applications se Wispr open karo
  • Pop-up aayega permissions ke liye, "Open System Settings" click
  • Microphone: Wispr toggle ON karo
  • Accessibility: Wispr toggle ON karo (text auto-paste ke liye zaroori)
2

Install + Permissions (Windows)

  • Downloaded .exe file run karo
  • Installer next-next karo, default settings rakho
  • Install hone pe Wispr open karo
  • Windows microphone permission popup aayega, "Allow" click
  • Sign in karo apne email se (free trial active ho jata hai)
3

Hotkey Set Karo

Default hotkey: Fn key. Press karke hold + bolo.

Default hotkey: Right Ctrl. Press karke hold + bolo.

Custom hotkey chahiye? Settings, Hotkey, koi bhi key choose karo.

Use Karne Ka Tarika

1

Cursor Wahan Rakho

Jis app mein likhna hai (WhatsApp, email, Claude, kuch bhi), us text box mein click karo. Cursor blink kar raha hona chahiye.

2

Hotkey Daboo, Bolo

Fn hold karo, bolna shuru karo:

Right Ctrl hold karo, bolna shuru karo:

  • Naturally bolo, jaise dost se baat kar rahe ho
  • "Comma", "full stop", "naya paragraph", bolne se punctuation aati hai
  • Roman Urdu mein bolo, automatically Roman likh deta hai
3

Hotkey Chhoro

Bolna khatam, hotkey release karo. 1-2 second mein text aa jaata hai jahan cursor tha.

Pehli baar awkward lagega apne aap se bolna. 2-3 din mein bilkul natural ho jata hai.

Wispr + Claude, Asli Power

  1. Browser mein claude.ai kholo
  2. Chat box mein cursor rakho
  3. Hotkey daboo, bolo apna sawaal
  4. Hotkey chhoro, text aata hai
  5. Enter, Claude jawab deta hai

Sirf 5 second ka kaam. Pehle 5 minute leta tha typing mein.

Pricing

Wispr Flow Mobile App

Wispr ka mobile app available hai. Mein khud iOS pe use karta hoon.

1

App Download Karo

  • iOS: App Store mein "Wispr Flow" search karo.
  • Android: Google Play Store mein "Wispr Flow" search karo.

Download Links →

2

Sign In + Permissions

  • Email se sign in karo (free trial active).
  • Microphone permission "Allow" karo.
  • Keyboard permission bhi enable karo (taake har app mein chal sake).
3

Wispr Keyboard Activate Karo

Phone settings mein jao:

  • iOS: Settings, General, Keyboard, Keyboards, Add New Keyboard, Wispr Flow.
  • Android: Settings, System, Languages & Input, On-screen keyboards, Wispr toggle on.

Phir keyboard jab khule, globe icon (iOS) ya keyboard switcher (Android) se Wispr select karo.

4

Use Karne Ka Tarika

  • Kisi bhi app mein typing area pe tap karo.
  • Wispr keyboard pe switch karo.
  • Mic icon press hold karo, bolo.
  • Release karte hi text aa jata hai.

WhatsApp, Email, Notes, har app mein chalega.

Note: Mobile app ka quality laptop wale ke barabar hai. Roman Urdu support same. Subscription dono pe sync hoti hai.
