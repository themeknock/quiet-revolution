Step 2, Apna Pehla Repository Banao

1

"+" Icon, "New repository"

GitHub mein top-right corner pe "+" icon hai, click karo, "New repository":

  • Repository name: apni app ka naam (e.g., my-website)
  • Public select karo (zaroori hai for free hosting)

Step 3, Apna Code Upload Karo

1

Drag & Drop, Sabse Easy

Repository page pe:

  • "Add file" button click
  • "Upload files" select
  • Apni HTML/CSS/JS files browser mein drag karo
  • Bottom mein commit message likho (e.g., "Initial upload")
  • "Commit changes" click

Code GitHub pe live ho gaya.

Note: Apni HTML file ka naam index.html rakhna zaroori hai. Yahi default page hota hai jo browser mein khulta hai.

Common Issues + Fixes

  • Site nahi khul rahi: Repo public hai? Settings mein check karo.
  • 404 Error: File ka naam index.html hai? Lowercase mein?
  • CSS load nahi ho rahi: File paths relative hain, absolute nahi (e.g., style.css not /style.css).
  • Update reflect nahi ho rahi: 1, 2 minute wait karo. Phir browser cache clear karo (Ctrl+Shift+R).
  • Custom domain not working: 10, 30 minute wait. DNS propagation slow hota hai.
