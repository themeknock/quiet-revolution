Phase 1, Setup (Ek Baar)

1

VS Code Install Karo

code.visualstudio.com pe jao, free download. Mac/Windows/Linux sab pe chalti hai.

Install hone ke baad kholo. Pehli baar dikhega clean editor.

2

Claude Code Extension

VS Code mein left sidebar pe "Extensions" icon (chautha icon). Search karo: Claude

"Claude Code" by Anthropic install karo. Yeh tumhe Claude se baat karne deta hai VS Code ke andar.

Note: Claude Code ke liye Claude Pro subscription ($20/mo) chahiye. Bina Pro ke yeh feature kaam nahi karega.

3

Wispr Flow Install

Apne system pe Wispr Flow install karo. Full setup guide yahan

Iska kaam, jab tumhara cursor kahin bhi ho (VS Code, Claude, Browser), tum bolo, text aa jaaye.

4

GitHub Account

Free signup at github.com. Yeh tumhara code save karne aur live deploy karne ke liye chahiye.

Phase 2, Daily Workflow

1

VS Code Kholo, Naya Folder

Apne Documents mein nayi folder banao, e.g. my-app. VS Code mein "File, Open Folder" karke woh folder kholo.

2

Claude Code Activate Karo

VS Code mein right side pe Claude icon dikhega ab. Click karo, Claude Code panel khulega.

Pehli baar login mango ga, Claude account se sign in karo.

3

Wispr Daboo, Apna Idea Bolo

Claude Code ke chat box mein cursor rakho. Wispr hotkey daboo, bolna shuru karo:

🎙️ Voice: Mujhe ek booking system banao salon ke liye. Customer name, phone, service, date, time slot. WhatsApp pe confirmation message bheje. Mobile-friendly green theme. Files yahin is folder mein banao.

Hotkey chhoro, text aa jaayega. Enter daboo.

4

Claude Code Khud Banata Hai

Claude Code khud:

  • Files banata hai (HTML, CSS, JS, ya Next.js project)
  • Code likhta hai
  • Folder structure set karta hai
  • Database connect karta hai agar zaroorat ho

Tum sirf dekhte ho. Files left sidebar mein appear hoti hain.

5

Test Karo Local Pe

Claude Code khud bata dega kaise run karna hai. Aam taur pe terminal mein:

$ npm install
$ npm run dev

Browser mein localhost:3000 open karo, app dikh jaayegi.

6

Adjust Karo (Bolne Se)

Kuch change chahiye? Wispr daboo, bolo:

🎙️ Voice: Time slots 30-30 minute ke karo. Logo upload ka option add karo. Confirmation message mein customer ka naam highlight karo.

Claude Code update kar deta hai apne aap.

Phase 3, Live Deploy Karna

1

Claude Code Ko Bolo, GitHub Pe Push Karo

Wispr daboo:

🎙️ Voice: Is project ka GitHub repo banao, code push karo, naam rakho salon-booking.

Claude Code:

  • Git initialize karta hai
  • GitHub pe new repo banata hai
  • Code push kar deta hai
2

GitHub Pages Enable Karo

GitHub pe apna repo kholo:

  • Settings tab
  • Pages (left sidebar)
  • Source: main branch, Save

30 second mein live ho jaata hai is URL pe: username.github.io/salon-booking

3

Bas, Live Hai!

Apna URL copy karo, client ko bhej do. App internet pe hai, kahin se bhi access ho sakti hai.

Future updates? Claude Code ko bolo: "Yeh changes karo aur push karo", app auto-update.

Common Confusions

"Claude Code aur Claude.ai mein farq kya hai?"

Claude.ai, browser version, basic chat ke liye. Free aur Pro dono.

Claude Code, VS Code extension, code likhne ke liye. Pro subscription chahiye. Yeh asli power hai.

"Pehle git, GitHub samjhna hai?"

Nahi. Claude Code khud handle karega. Tum sirf bolo "GitHub pe push karo." Yeh terminal commands khud chala lega.

"Errors aaye toh kya karoon?"

Error message Claude Code ko paste karo, ya Wispr se bolo: "Yeh error aa raha hai, fix karo." Solve kar deta hai.

"Multiple projects parallel mein?"

VS Code mein different folders kholo. Each folder = different project = different GitHub repo = different live URL.

"Phone se ho sakta hai?"

Phone pe VS Code chalna mushkil hai. Laptop required hai is workflow ke liye. Phone se sirf Claude.ai chat use kar sakte ho.
