Prompt Engineering, Kya Achi Image Banata Hai

Bad Prompt

A man working

Result: generic, boring, kahin bhi ka aadmi ho sakta hai.

Good Prompt

A young Pakistani entrepreneur (25 years old) working on a laptop at a wooden cafe table in Lahore, late afternoon golden hour light streaming through window, plants in background, blue shirt, focused expression. Cinematic photography. Sony A7 IV, 50mm lens, f/1.8. Warm color grade.

Result: specific, atmospheric, professional.

Magic Formula

Achi prompt ka structure:

  1. Subject: Kaun (age, ethnicity, clothing)
  2. Action: Kya kar raha hai
  3. Setting: Kahan hai (location, mood)
  4. Lighting: Kaisi roshni
  5. Camera: Lens type, aperture (cinematic feel)
  6. Style: Photo / illustration / 3D / anime
  7. Color grade: Warm / cool / specific

Pakistan-Specific Image Generation

Pakistani context ke liye prompts:

  • Locations: Lahore rooftop, Karachi beach, Islamabad cafe, Multan bazaar
  • People: South Asian / Pakistani features specify karo
  • Clothing: "shalwar kameez", "kurta", "western casual"
  • Cultural elements: chai, charpai, rickshaw, mosque, gulmohar tree

AI ko bilkul specific batao, generic global imagery se bachno.

Workflow, Client Order

  1. Client se brief lo, kya chahiye, kis style mein
  2. 3-4 prompt variations Claude se likhwao
  3. Flow / Midjourney mein generate karo
  4. Best 5 images select karo
  5. Photoshop / Canva mein touch-up (agar zaroorat ho)
  6. Client ko deliver

Total time per image: 5-15 minute. Pehle 1-2 ghante leta tha photographer ke saath.

Pricing, Fiverr / Local

  • Fiverr Single Image: $10-30
  • Fiverr Package (5 images): $40-80
  • Local Pakistani Brand (10 images/mo): Rs. 15,000-30,000
  • Custom Photoshoot Replacement: $200-500 per project

Tips, Stand Out

  • Reference images bhejo client ko approve karne ke liye pehle
  • 3 variations dao har request mein
  • High-resolution deliver karo (4K minimum)
  • PSD/editable file bhi do (extra fee charge kar sakte ho)
  • Specialize karo, sirf food, sirf fashion, sirf real estate, etc.
